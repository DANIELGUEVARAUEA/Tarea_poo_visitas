# Sistema de Registro de Visitantes

## Descripción

Aplicación de escritorio desarrollada en Python con graficos de Tkinter.  
Permite registrar, mostrar y eliminar visitantes usando una arquitectura modular por capas (Modelo, Servicio, UI y Main).  
Los datos se guardan en un archivo `visitantes.txt` y se registra automáticamente la fecha y hora del computador con datetime.

El programa utiliza Programación Orientada a Objetos, inyección de dependencias y encapsulamiento, sin librerías externas.
